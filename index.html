Work

pastemyst.java

An API wrapper for the paste.myst.rs code sharing website, providing full coverage of all API endpoints.


Conways Game Of Life

An implementation of John Conway's cellular automaton The Game of Life, using C# and Unity for the graphics. Running it only reproduces one pattern, but opening it in Unity allows you to customize the patterns.


Pomodoro Timer

An iOs pomodoro timer that I initially made just playing around with Swift, but somehow turned into a pretty functional app!


Covid.NET

During the Covid 19 pandemic, I made this API wrapper to get information on Covid 19. This was initially to, again, play with C#, but with some help from my friends it turned into a decent project!



Govid

A Golang library that serves the same purpose as Covid.NET, just...in Go!


Random Facts!

A Svelte application that serves the most random facts if you ever need them for whatever you may need random facts for.


Orbital

A Java based game framework with a functioning renderer, input system, and an ECS architecture designed to make it simple for people to build games quickly, using Raylib under the hood.


Zeitplan

An iOs app written for the Swift Student Challenge, to help people with dementia/disassosiative disorders plan out their days. Works on iPad and iPhone.
